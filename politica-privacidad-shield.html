GuardianOS Shield — Control Parental

Protección infantil con filtrado DNS y monitoreo de apps. Todos los datos permanecen en tu dispositivo.

✓ Privacidad garantizada: GuardianOS Shield almacena todos los datos localmente en tu dispositivo. No hay servidores externos ni seguimiento de terceros. El filtrado DNS usa CleanBrowsing pero NO inspecciona el contenido del tráfico.

2. Datos que accede la app

GuardianOS Shield necesita acceder a cierta información de tu dispositivo para proporcionar control parental efectivo. Toda esta información permanece en tu dispositivo.

📱 Información del dispositivo

  • Modelo y fabricante (para optimización de servicios)
  • Versión de Android (compatibilidad de funciones)
  • Estado de batería (excluir de optimizaciones)

📦 Lista de aplicaciones instaladas

  • Nombre de las aplicaciones
  • Nombre del paquete (package name)
  • Tiempos de uso de cada app (UsageStats)
  • Apps detectadas en primer plano (para bloqueo parental)

🌐 Consultas DNS bloqueadas

  • Dominios bloqueados por filtro (ej: sitio-adulto.com)
  • Fecha y hora del intento de acceso
  • Categoría del bloqueo (adulto, redes sociales, malware)
  • NO se almacena: Contenido de páginas, credenciales, tráfico cifrado

🔒 Configuración de control parental

  • PIN parental (encriptado con AndroidKeyStore)
  • Horarios de uso permitidos
  • Nivel de restricción configurado
  • Apps marcadas como sensibles por padres

3. Qué NO hacemos con tus datos

  • NO enviamos datos a servidores externos (excepto DNS a CleanBrowsing)
  • NO vendemos información
  • NO compartimos con terceros
  • NO inspeccionamos contenido de tráfico HTTPS
  • NO rastreamos ubicación
  • NO accedemos a mensajes, fotos o archivos personales

4. Almacenamiento de datos

Los datos se guardan localmente:

  • Base de datos Room: Historial de sitios bloqueados, perfiles de usuario, configuración (almacenado en /data/data/com.guardianos.shield/databases/)
  • DataStore encriptado: Configuración de la app y estado de servicios
  • EncryptedSharedPreferences: PIN parental hasheado con SHA-256 y encriptado con AES256-GCM
  • Retención: Logs de DNS se eliminan automáticamente después de 30 días (configurable)

5. Filtrado DNS con CleanBrowsing

¿Qué es CleanBrowsing?

CleanBrowsing es un servicio de DNS público que filtra contenido inapropiado. GuardianOS Shield redirige consultas DNS a sus servidores (185.228.168.168) cuando el modo VPN está activo.

Datos enviados a CleanBrowsing:

  • Nombres de dominio consultados (ej: google.com, facebook.com)
  • Dirección IP del dispositivo (temporal, para responder la consulta)

Política de CleanBrowsing: https://cleanbrowsing.org/privacy

Importante: CleanBrowsing NO puede ver el contenido cifrado de tus conexiones HTTPS. Solo ve los nombres de dominio.

6. Exportar estadísticas

GuardianOS Shield permite exportar informes en CSV/JSON:

  • Se genera un archivo con las estadísticas de la app
  • TÚ decides con quién compartirlo (email, Drive, etc.)
  • Usamos el sistema de compartir de Android
  • Los archivos exportados contienen datos sensibles de uso de apps
⚠️ Importante: Si compartes un informe CSV/JSON, este contendrá información sobre apps usadas y sitios bloqueados. Comparte solo con personas de confianza.

7. Permisos solicitados

La app solicita los siguientes permisos de Android:

🔍 QUERY_ALL_PACKAGES

¿Para qué? Permite listar todas las aplicaciones instaladas para monitoreo de control parental.

Justificación: Esencial para detectar y bloquear apps sensibles según la configuración de padres/tutores.

📊 PACKAGE_USAGE_STATS

¿Para qué? Permite detectar qué app está en primer plano y medir tiempos de uso.

Justificación: Necesario para aplicar horarios de uso y bloquear apps fuera del horario permitido.

🔒 BIND_VPN_SERVICE

¿Para qué? Permite crear un túnel VPN local para interceptar consultas DNS.

Justificación: Es el mecanismo que permite el filtrado DNS sin inspeccionar el tráfico de datos.

Aclaración: La VPN es local, NO redirige tu tráfico a servidores remotos. Solo redirige DNS a CleanBrowsing.

🔔 POST_NOTIFICATIONS (Android 13+)

¿Para qué? Mostrar notificaciones cuando se bloquea un sitio o app.

🔋 REQUEST_IGNORE_BATTERY_OPTIMIZATIONS

¿Para qué? Evitar que el sistema mate los servicios de monitoreo en segundo plano.

Justificación: Necesario para protección continua sin interrupciones.

8. Conexión a Internet

GuardianOS Shield usa Internet solo para:

  • ✅ Consultas DNS a CleanBrowsing (cuando VPN activo)
  • ✅ Futuras actualizaciones de listas de bloqueo (opcional)

NO usamos Internet para:

  • ❌ Enviar estadísticas de uso
  • ❌ Analytics o seguimiento de usuarios
  • ❌ Publicidad
  • ❌ Sincronización en la nube

9. Terceros y servicios externos

Servicios externos utilizados:

  • CleanBrowsing DNS: Filtrado de contenido (ver sección 5)

NO integrados:

  • ❌ Google Analytics / Firebase Analytics
  • ❌ Redes publicitarias
  • ❌ SDKs de redes sociales
  • ❌ Servicios de crash reporting externos

10. Niños y menores

GuardianOS Shield está diseñado específicamente para proteger a menores bajo la supervisión de padres/tutores. Dado que:

  • Los datos permanecen en el dispositivo local
  • No hay registro ni cuentas de usuario
  • Los padres controlan completamente la configuración

Cumplimos con COPPA (Children's Online Privacy Protection Act) y RGPD en lo relativo a protección de menores.

11. Tus derechos (RGPD)

Bajo el RGPD, tienes derecho a:

  • Acceso: Todos los datos están en tu dispositivo (base de datos local)
  • Rectificación: Modifica configuración desde la app
  • Supresión: Usa "Limpiar historial" en Configuración o desinstala la app
  • Portabilidad: Exporta tus datos en CSV/JSON
  • Oposición: Desactiva funciones específicas en Configuración
  • Contacto: info@guardianos.es para cualquier solicitud

12. Seguridad de los datos

Medidas de seguridad implementadas:

  • Encriptación: PIN parental con AES256-GCM + SHA-256
  • Almacenamiento local: Datos no salen del dispositivo
  • Sin backups en la nube: allowBackup="false" en AndroidManifest
  • Código abierto: Disponible en GitHub para auditoría pública
